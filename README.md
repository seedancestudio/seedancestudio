# Seedance Studio

[Seedance Studio](https://seedancestudio.org/) is an AI video creation studio for turning text prompts and images into short videos with Seedance 2.0, Seedance 1.5 Pro, Kling, and other production-ready video models.

Website: [seedancestudio.org](https://seedancestudio.org/)

Workflows include text-to-video concepts, image-to-video clips, reference-based video generation, video editing, and video extension.
